# Slack Pull

This directory is a placeholder for any custom Slack scripts you might need.

The vault's Slack integration uses the Slack MCP server directly -- no local scripts required. The `/pull-slack` command in `.claude/commands/pull-slack.md` handles reading and dispatching messages via MCP tools.

If your Slack workspace requires custom auth or token management, put those scripts here.

## Setup

See [docs/mcp-setup.md](../../docs/mcp-setup.md) for Slack MCP server configuration.
